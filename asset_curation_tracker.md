## Category Routing Logic

Assets are categorized by the `extractLocation()` function in `App.jsx`:

1. **MM_ series** — Manual routing based on content:
   - Parks: MM11, MM19, MM22, MM40
   - Urban Streets: MM13 (Subway), MM21 (Bike)
   - Rooftops: MM03, MM08, MM18, MM44, MM45, S86_Socks
   - Creative Studios: MM02, MM07, MM12, MM24, MM27, MM42
   - Fallback: Rooftops

2. **S_ series** — Automatic routing via filename patterns:
   - `UrbanStreets` or `UrbanNight` → Urban Streets
   - `Cafes`, `Bar`, `Matchbook`, `CafeScene`, `CafeWindow` → Cafés
   - `Studios`, `Flatlay`, `Laydown`, `Desk` → Creative Studios
   - `Rooftops`, `Skyline`, `Silhouette`, `Sunset`, `CityView` → Rooftops
   - `Parks`, `Nature`, `Grass`, `Natural` → Parks

3. **V_ videos** — Same pattern matching as images

4. **Product fallback** — Items with Bandana, Socks, Tote, Hoodie, etc. → Creative Studios
